# 月球矿车路线解谜游戏

一款基于 Phaser 3 + React + TypeScript 开发的月球矿车路线解谜游戏。玩家驾驶小矿车在月面基地之间运输矿石，需要连接采矿点、熔炼站、氧气塔、太阳能充电区和返回舱。

## 原始需求

> 请用 Phaser 制作月球矿车路线解谜游戏，玩家驾驶小矿车在月面基地之间运输矿石，需要连接采矿点、熔炼站、氧气塔、太阳能充电区和返回舱。矿车能切换轨道、短距离跃迁、牵引货箱和手动刹车，低重力会放大落点偏差与制动距离；每关目标是在有限电量、陨石干扰、货箱重量、轨道岔路和返回窗口之间规划路线。操作反馈要清楚，玩家失败时能分辨是电量耗尽、货箱脱轨、撞上障碍、错过发射窗口还是矿石价值不足，结算要让人想重新优化路线。

## 游戏特色

- 🚗 **低重力物理**：月球低重力环境，制动距离更长，需要提前规划
- ⛏ **多种基地**：采矿点、熔炼站、氧气塔、太阳能区、返回舱
- 🔀 **轨道岔路**：多个岔路口，选择不同路线影响效率
- ⚡ **短距跃迁**：消耗电量进行短距离瞬移，避开陨石
- 📦 **货箱系统**：可挂载/卸下载货箱，影响速度和耗电
- ☄ **陨石干扰**：定时掉落的陨石，注意红色警告区域
- ⏱ **返回窗口**：限时返回，考验路线规划能力
- 🏆 **三星评价**：追求最优路线，获得三星评价

## 启动方式

### 前置要求

- Node.js >= 18
- npm 或 pnpm

### 本地开发启动

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务器

```bash
npm run dev
```

访问地址：http://localhost:5173

#### 3. 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### Docker 一键启动

#### 前置要求

- Docker
- Docker Compose

#### 启动命令

```bash
docker compose up --build
```

后台运行：

```bash
docker compose up --build -d
```

访问地址：http://localhost:8080

停止服务：

```bash
docker compose down
```

## 操作说明

| 按键 | 功能 |
|------|------|
| W / ↑ | 加速前进 |
| S / ↓ | 刹车减速 |
| A / ← / Q | 岔路口选择左侧轨道 |
| D / → / E | 岔路口选择右侧轨道 |
| SPACE | 短距离跃迁（消耗电量） |
| E | 挂载/卸下载货箱 |

也可以点击屏幕下方的控制按钮进行操作。

## 基地类型

- ⛏ **采矿点**：装载矿石，增加价值与重量
- 🔥 **熔炼站**：将已有矿石价值乘以倍率
- 💨 **氧气塔**：补充氧气储备
- ☀ **太阳能区**：补充电量储备
- 🚀 **返回舱**：终点，需满足最低矿石价值
- 🏁 **起点**：矿车出发点

## 失败原因

- 🔋 **电量耗尽**：电量归零且矿车已停下
- 💨 **氧气耗尽**：氧气储备归零
- 📦 **货箱脱轨**：牵引货箱时高速急转弯
- 💥 **撞击障碍**：高速撞上月岩或陨石
- ⏰ **错过窗口**：返回倒计时结束未能抵达
- ⛏ **价值不足**：抵达返回舱但矿石价值不够

## 关卡说明

游戏包含 3 个关卡，难度递增：

1. **第一关：新手训练** - 熟悉操作，完成简单采矿和返回
2. **第二关：熔炼增值** - 学习使用熔炼站提升矿石价值
3. **第三关：终极挑战** - 多采矿点、多岔路、陨石密集

## 策略提示

- 低重力环境下制动距离很长，提前刹车！
- 货箱越重，最高速度越低，耗电越快
- 跃迁可以越过陨石警告区，节省时间
- 太阳能区和氧气塔可重复进入补充
- 熔炼站需先装矿石才能生效
- 想拿三星？尽量节省资源并快速返回！
